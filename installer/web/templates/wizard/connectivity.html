Onion Routing

The host runs a Tor daemon. Each VM gets its own .onion hidden service. Subscribers connect via Tor: torsocks ssh user@xxx.onion.

No broker, no IPv6, no TLS. Tor provides end-to-end encryption. Subscribers will need Tor installed locally.
